# Async-Selector-Kit Documentation

Created with Gastby

## Getting Started

1. git clone
2. yarn
3. `yarn develop`

## Publishing

### GitHub Actions

- pull-requests and pushes will build and deploy to the `gh-pages` branch

### Manually

1. `git checkout docs` - make sure you're on docs branch
2. `yarn build`
3. `yarn deploy`
